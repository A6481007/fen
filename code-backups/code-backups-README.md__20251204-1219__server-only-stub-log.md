## Naming convention
- Use <path-with-separators-replaced>__YYYYMMDD-HHMM__<short-change-label>.<ext> when copying files into `code-backups/`.
- Record a short reason plus an explicit restore command for every backup. For brand-new files with no prior version, note that rollback is deleting the new file.

## Raw logs
- Dated folders (e.g., 27-11-2025-15-02/README.md, 20251130-1608/README.txt) keep the original notes; entries below consolidate them without removing the dated files.
